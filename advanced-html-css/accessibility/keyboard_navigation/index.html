Semantic HTML

All elements are semantic HTML elements and are focusable in order. 

          
            <div class="buttons">
              <button>1</button>
              <button>2</button>
              <button>3</button>
            </div>

Semantic and non-semantic HTML

There is only 1 semantic HTML element, 2 non-semantic HTML elements, and are all focusable in order. 

          
            <div class="buttons">
              <button>1</button>
              <div tabindex="0">2</div>
              <div tabindex="0">3</div>
            </div>
2
3

Semantic and non-semantic HTML

There is only 1 semantic HTML element, 2 non-semantic HTML elements, and are all focusable not in order. 

          
            <div class="buttons">
              <button>1</button>
              <div tabindex="5">2</div>
              <div tabindex="0">3</div>
            </div>
2
3

Semantic and non-semantic HTML

There is only 1 non-focusable semantic HTML element, and 2 focusable non-semantic HTML elements, 

          
            <div class="buttons">
              <button tabindex="-1">1</button>
              <div tabindex="0">2</div>
              <div tabindex="0">3</div>
            </div>
