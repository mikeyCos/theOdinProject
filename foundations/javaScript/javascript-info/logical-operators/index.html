What's the result of OR?

  1. What is the code below going to output?

What's the result of OR'ed alerts?

  1. What will the code below output?

What is the result of AND?

  1. What is this code going to show?
    • alert( 1 && null && 2 );

What is the result of AND'ed alerts?

  1. What will this code show?
    • alert( alert(1) && alert(2) );

The result of OR AND OR

  1. What will the result be?
    • alert( null || 2 && 3 || 4 );

Check the login

  1. Write the code which asks for a login with prompt.
  2. If the visitor enters "Admin", then prompt for a password, if the input is an empty line or Esc – show “Canceled”, if it’s another string – then show “I don’t know you”.
  3. The password is checked as follows:
    • If it equals “TheMaster”, then show “Welcome!”,
    • Another string – show “Wrong password”,
    • For an empty string or cancelled input, show “Canceled”
  4. Please use nested if blocks. Mind the overall readability of the code.
  5. Hint: passing an empty input to a prompt returns an empty string ''. Pressing ESC during a prompt returns null.
