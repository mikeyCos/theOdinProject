Is "else" required?

  1. The following function returns true if the parameter age is greater than 18.
  2. Otherwise it asks for a confirmation and returns its result:
    3.                 
                function checkAge(age) {
                    if (age > 18) {
                        return true;
                    } else {
                        // ...
                        return confirm('Did parents allow you?');
                    }
                }
  3. Will the function work differently if else is removed?
    4.                 
                function checkAge(age) {
                    if (age > 18) {
                        return true;
                    }
                    // ...
                    return confirm('Did parents allow you?');
                }
  4. Is there any difference in the behavior of these two variants?

Function min(a, b)

  1. Write a function min(a,b) which returns the least of two numbers a and b.
  2. For instance:
    3.                 
                min(2, 5) == 2
                min(3, -1) == -1
                min(1, 1) == 1

Function pow(x,n)

  1. Write a function pow(x,n) that returns x in power n. Or, in other words, multiplies x by itself n times and returns the result.
    2.                 
                pow(3, 2) = 3 * 3 = 9
                pow(3, 3) = 3 * 3 * 3 = 27
                pow(1, 100) = 1 * 1 * ...* 1 = 1
  2. Create a web-page that prompts for x and n, and then shows the result of pow(x,n).
  3. P.S. In this task the function should support only natural values of n: integers up from 1.
