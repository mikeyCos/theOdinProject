                let i = 3;

Which values does the while loop show?

  1. For every loop iteration, write down which value it outputs and then compare it with the solution.
  2. Both loops alert the same values, or not?
    • The prefix form ++i:
                    
                let i = 0;
                while (++i < 5) alert( i );
    • The postfix form i++
                    
                let i = 0;
                while (i++ < 5) alert( i );

Which values get shown by the "for" loop?

  1. For each loop write down which values it is going to show. Then compare with the answer.
  2. Both loops alert same values or not?
    • The postfix form:
                    
                for (let i = 0; i < 5; i++) alert( i );
    • The prefix form:
                    
                for (let i = 0; i < 5; ++i) alert( i );

Repeat until the input is correct

  1. Write a loop which prompts for a number greater than 100. If the visitor enters another number – ask them to input again.
  2. The loop must ask for a number until either the visitor enters a number greater than 100 or cancels the input/enters an empty line.
  3. Here we can assume that the visitor only inputs numbers. There’s no need to implement a special handling for a non-numeric input in this task.

Output prime numbers

  1. An integer number greater than 1 is called a prime if it cannot be divided without a remainder by anything except 1 and itself.
  2. In other words, n > 1 is a prime if it can’t be evenly divided by anything except 1 and n.
  3. For example, 5 is a prime, because it cannot be divided without a remainder by 2, 3 and 4.
  4. Write the code which outputs prime numbers in the interval from 2 to n.
  5. For n = 10 the result will be 2,3,5,7.
  6. P.S. The code should work for any n, not be hard-tuned for any fixed value.
